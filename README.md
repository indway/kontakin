# Kontakin

> Easy way to connect without saving contacts to your phone.

Kontakin adalah aplikasi sederhana untuk menghubungkan orang tanpa perlu menyimpan kontak di perangkat.   
Cocok untuk use-case cepat seperti networking, sharing nomor sekali pakai, atau interaksi sementara.

## 🚧 Status

Project ini masih dalam tahap **Work In Progress (WIP)**.
Beberapa fitur mungkin belum stabil atau masih berubah.

## ✨ Features (Planned & Available)

* 📇 View contact details
* ➕ Add & edit contact
* 📞 Keypad dialer
* 📋 Contact lists
* 🕓 Recent contacts
* 📷 QR scan contact
* ⚙️ Settings, import & sync

## 🤝 Contributing

Kontribusi sangat terbuka! Kalau kamu tertarik bantu develop, improve UI/UX, atau kasih ide, silakan ikut 🙌

### Cara kontribusi

1. Fork repository ini
2. Buat branch baru:

   ```bash
   git checkout -b feature/nama-fitur
   ```
3. Commit perubahan kamu:

   ```bash
   git commit -m "feat: tambah fitur xyz"
   ```
4. Push ke branch kamu:

   ```bash
   git push origin feature/nama-fitur
   ```
5. Buat Pull Request 🚀

## 📌 Guidelines

* Gunakan commit message yang jelas (disarankan pakai conventional commits: `feat`, `fix`, `docs`, dll)

* Untuk perubahan besar, diskusikan dulu lewat issue
